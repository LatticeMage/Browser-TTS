# Browser-TTS


## Requirement

* Win10 have japanese lang installed
* Browser driver such gecko
* env 
```
pip install -r requirements.txt
```

## Usage

Setting file ```config.ini``` :

```
[Settings]
gecko_driver_path = <path-to-gecko>
read_url = <url you want read>
```

Start

```
python src/main.py
```
